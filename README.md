# The Heritage

An "Old Money" aesthetic React Native Expo application.

## Features

- **The Moving Portraits (Reels):** A vertical video feed with a cinematic, art gallery feel.
- **The Governance (Settings):** A settings panel styled like a private club management interface.
- **The Personal Archive (Profile):** A profile view tracking consistency and collected wisdom.

## Tech Stack

- React Native
- Expo (Expo Router)
- TypeScript
- Google Fonts (Libre Baskerville, Cormorant Garamond, Libre Caslon Text)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npx expo start
   ```

3. Scan the QR code with your phone (using Expo Go) or run on a simulator.

## Design System

- **Colors:** British Racing Green, Cream/Parchment, Oxblood, Champagne Gold.
- **Typography:** Serif fonts for a classic, timeless look.
